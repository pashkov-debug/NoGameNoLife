Целевая модель MVP

Клиент:

PWA в браузере
один экран локации
лог событий
форма ввода действия
панель персонажа


Сервер:

API слой
world service
rule engine
AI gateway
stream layer

Хранилища:

PostgreSQL
Redis
object storage для изображений локаций

Главный принцип:
AI не управляет состоянием мира.
AI интерпретирует и рассказывает.
Состояние меняет только rule engine.
